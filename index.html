Core Value

不是让 AI 变透明，而是让数据处理可审计

保护原始数据

默认不覆盖 data/raw/，处理结果另存到 results、figures、reports 和 audit。

记录处理过程

生成分析计划、处理日志和自检报告，记录每一步输入、输出、参数和数据规模变化。

自动表格自检

内置脚本检查文件哈希、行列数、缺失值、重复行、数值列、常量列和高缺失列。

追踪图表来源

每张图都说明来源表、生成脚本、绘图参数、标签完整性和解释风险。

Install

安装方式

下载压缩包后，把其中的 .agents/skills/bio-data-audit/ 放到你的 Codex 项目根目录。然后在 Codex 里使用 $bio-data-audit 调用。 

你的项目/
└── .agents/
    └── skills/
        └── bio-data-audit/
            ├── SKILL.md
            ├── agents/openai.yaml
            └── scripts/basic_table_audit.py

Bio Data Audit 是一个用于 Codex 的生物数据分析审计 skill。它不试图暴露 AI 内部推理，而是要求每次数据处理留下可复核证据：输入文件、行列数、过滤标准、 归一化方法、统计参数、图表来源、输出文件和潜在风险。
